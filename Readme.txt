For matlab version 9.4 (R2018a) or higher

English part:
There are three different scene search algorithms that I have implemented.
Full Enumeration Algorithm (GoodWay.m)
Artificial Bee Colony Algorithm (GoodWay_Roy.m)
Potential Fields Algorithm (GoodWay_pole.m)
They are used together with algorithms for creating a virtual scene in the main function test.m
The potential field algorithm turned out to be the best


Русская часть:
Есть три различных алгоритма поиска пути в сцене, которые я реализовала.
Алгоритм полного перебора (GoodWay.m)
Алгоритм пчелинного роя (GoodWay_Roy.m)
Алгоритм потенциальных полей (GoodWay_pole.m) 
Они используются вместе с алгоритмами создания виртуальной сцены в основной функции test.m
Алгоритм потенциальных полей оказался наилучшим
